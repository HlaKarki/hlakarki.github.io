After that, it was easier forward. The biggest problem I faced during this assignment was during the filter part. I was initially using only one array which was fixed; the global state array. So, I was going through the filter from top to bottom; i.e. text, price, city, condition. With too many nested if statements. I also learned how to create tags such as "a" tags and "span" tags and how i could use setAttribute and textContext to add href, links and texts.
